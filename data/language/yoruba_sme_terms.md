# Yorùbá SME Terms — AfrekaOS Offline

> Ìràwọ̀ àwọn ọ̀rọ̀ òṣòwọ́ fún AfrekaOS. Public, challenge-safe. No private data.

## Common SME operations terms (Yorùbá)

- **Inventory / Ẹ̀rọ̀jà** — àwọn ọjà tó wà nínú títà.
- **Supplier / Olùpèsè** — ẹni tó ń rán tàbí ń ta ọjà fún ìdájọ́.
- **Customer / Onírànwọ́-tà-bà-bà / Alábarà** — ẹni tó ń ra ọjà.
- **Credit / Gbèsè** — kí ẹnìkan gba ọjà kí ó sì san owó nìtókè.
- **Cashflow / Ìrìn owó** — bí owó ṣe ń wọ̀ àti tí jáde nínú iṣẹ́.
- **Records / Àkọsílẹ̀** — ìkósílẹ̀ tàbí àkánti tó ń ṣe ìtọ́ni.
- **Stockout / Àìsí ọjà** — nígbà ọjà tó ń lọ láìsí èyí tó tún wà.
- **Expansion / Ìfẹsẹ̀múlẹ̀** — fẹ̀ iṣẹ́ sílẹ̀ títun tàbí gíga rẹ̀.
- **Staff / Àwọn ọmọ-iṣẹ́** — àwọn ènìyàn tó ń ṣiṣẹ́ nínú ìdájọ́.
- **Daily checklist / Àkójọpọ̀ iṣẹ́ ojóumọ́** — àkójọ àwọn iṣẹ́ láti ṣe lójúmọ́.

## Fallback note

Ọ̀rọ̀ tó nira láti túmọ̀ lè ṣètúrójáde nínú èdè Gẹ̀ẹ́sì fún ìmọ̀ràn.

## Cautionary boundary language

- AfrekaOS ń ṣèrànwọ́ lórí ìmúṣẹ̀ iṣẹ́ nìkan.
- Kì í ṣe ètò ìṣirò owó, bánkì, iṣẹ́-ṣíṣe owó, owó-orí, tàbí ERP.
- Ṣàyẹ̀wò àkọsílẹ̀ rẹ kí o tó bẹ̀rẹ̀ sí ṣe ohó kan.
